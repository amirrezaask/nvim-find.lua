- buffers 
- diagnostics 
- diagnostics_buffer 
- files 
- git_files 
- helptags 
- live_ripgrep (WIP)
- lsp_document_symbols 
- lsp_workspace_symbols 
- oldfiles 
- ripgrep_fuzzy 
- ripgrep_qf
