# Nvim-Finder
Set of different finders for neovim, focues on simplicity.

- buffers 
- diagnostics 
- diagnostics_buffer 
- files 
- git_files 
- helptags 
- live_ripgrep (WIP)
- lsp_document_symbols 
- lsp_workspace_symbols 
- oldfiles 
- ripgrep_fuzzy 
- ripgrep_qf 


## Default Keybindings
*<C-n> / <Down>* => Next entry,
*<C-p> / <Up>* => Prev entry,
*<C-y> / <Enter* => Accept Entry,
*<C-d>* => Page Down,
*<C-u>* => Page Up
